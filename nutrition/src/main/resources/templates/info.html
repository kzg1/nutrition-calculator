Info:


Percentages:
  • If the chosen percentage is 100, only the first food will be calculated.
  • If the chosen percentage is between 0 and 100, the result will be calculated for both food.
  • If the percentage is below 0 or above 100, the system takes it as 100% for the first food.
Foods:
  • If you choose the same food for multiple nutrition source, you have to consume the aggregate value.
  • If you want to add/modify a food, you have to add its parameters to the foods.csv
Nutritions:
  • If the chosen nutrition values impossible to calculate using the selected foods and percentages you won't get values.
